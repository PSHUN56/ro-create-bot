# Ro Create Bot

Русскоязычный Discord-бот для сервера разработчиков Roblox Studio. В боте уже есть:

- русские slash-команды;
- экономика с монетами;
- `/daily` и `/work`;
- ежедневные задания с отправкой скриншота на проверку;
- модерация заданий через кнопки `Принять` и `Отклонить`;
- DM-уведомления участнику после решения;
- публикация объявлений за 2000 монет;
- автосоздание ролей и приватных тестовых каналов;
- только канал `новости` создается открытым, остальные закрыты для теста.

## Что умеет бот

### Экономика

- `/balance` - баланс пользователя.
- `/profile` - профиль пользователя.
- `/daily` - ежедневный бонус.
- `/work` - подработка с кулдауном.
- `/add-coins` - ручная выдача монет админом.

### Ежедневные задания

- `/tasks` - показывает сегодняшнее задание.
- `/submit-task` - отправляет задание на проверку со скриншотом и описанием.
- В канал `проверка-заданий` приходит embed с кнопками.
- Проверяющий может принять или отклонить задание.
- При отклонении открывается окно для комментария.
- Бот отправляет участнику ЛС с результатом.

### Объявления

- `/post-ad` - отправка объявления на модерацию.
- Стоимость публикации по умолчанию: `2000` монет.
- Если объявление отклонено, монеты возвращаются.
- После одобрения объявление публикуется в канал `объявления`.

## Какие каналы и роли создаются

Роли:

- `Основатель`
- `Администрация`
- `Проверяющий задания`
- `Модератор объявлений`
- `Разработчик`
- `Скриптер`
- `Билдер`
- `UI-дизайнер`
- `3D-моделлер`
- `Аниматор`

Категории и каналы:

- `Информация`
- `новости` - открыт для всех
- `обновления-бота` - приватный
- `Экономика`
- `работа-и-монеты` - приватный
- `ежедневные-задания` - приватный
- `отправка-заданий` - приватный
- `Биржа Ro Create`
- `объявления` - приватный
- `портфолио` - приватный
- `devlog` - приватный
- `Штаб модерации`
- `проверка-заданий` - приватный
- `проверка-объявлений` - приватный

После команды `/setup-server` бот создает недостающие роли и каналы. Каналы делаются приватными для теста и видны staff-ролям и пользователю, который запускал настройку.

## Запуск

1. Установи Node.js 20+.
2. Скопируй `.env.example` в `.env`.
3. Заполни:

```env
DISCORD_TOKEN=токен_бота
CLIENT_ID=id_приложения
GUILD_ID=id_сервера
ANNOUNCEMENT_COST=2000
```

4. Установи зависимости:

```bash
npm install
```

5. Запусти бота:

```bash
npm start
```

## Что важно для Discord

Боту нужны права:

- `Manage Roles`
- `Manage Channels`
- `Send Messages`
- `Use Slash Commands`
- `Read Message History`
- `Manage Messages`

Для DM-уведомлений у участника должны быть открыты личные сообщения от участников сервера.
